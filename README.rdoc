This project was intended as a quick exorcise to compare using MYSQL over Postgres. 

As a user I want to be able to create an account sign in and log out.

As a user I want to be able to add a new adventure.

As a user I want to be able to search for an adventure by category and location.

As a user I want to be able to leave a review of an adventure.

As a user I want to be able to 'purchase' or 'go on an adventure'
